Privacy Policy — FanTalk (Test Environment)

Last updated: December 9, 2025

This privacy policy applies to the *test environment* used to validate Meta (WhatsApp) integrations for the FanTalk platform. This is not the official production application or its final privacy policy.

1. Information We Collect

In this test environment, we may temporarily collect:

  • Basic user identifiers (name, phone number, email).
  • Message metadata and message content exchanged using WhatsApp API.
  • Device and browser information, logs, and IP address.

2. Purpose of Data Usage

Data is used strictly to:

  • Test and validate WhatsApp API functionality.
  • Ensure message delivery and platform behavior.
  • Identify and resolve integration or technical issues.

No data is sold or used for marketing purposes.

3. Data Sharing

Data may be shared only with:

  • Meta/WhatsApp services for message processing.
  • Infrastructure providers used to host this test environment.
  • Legal authorities if required by law.

4. Data Retention & Deletion

All test data may be deleted at any time upon request. Users may request data deletion via:

contato@fantalk.com.br

5. Security

Even in a test environment, we apply reasonable technical measures such as HTTPS, authenticated access, and secure storage.

6. Policy Updates

This temporary policy may be updated as the integration tests evolve. Any changes will appear on this page.
